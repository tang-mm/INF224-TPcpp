Detailed Description

Class for exchanging data blocks or text lines between SOCK_STREAM sockets. SOCK_STREAM (i.e. TCP/IP) sockets do not preserve record boundaries. Messages can thus be split or merged so that one call to Socket::send() on the sending side does not necessarily correspond to one call to Socket::receive() on the receiving side. This class makes it easier to solve this problem by providing functions that call send() or receive() as many times as needed.
