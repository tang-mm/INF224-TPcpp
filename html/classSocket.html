Detailed Description

TCP/IP or Datagram socket. This class supports TCP/IP or Datagram AF_INET connections (IPv4 Internet protocol). Note: other families, such as AF_INET6 or AF_UNIX not supported.

Member Enumeration Documentation

enum Socket::Errors

Socket errors.

  • Socket::FAILED (-1): could not connect
  • Socket::UNKNOWN_HOST (-2): could not reach host
  • Socket::INVALID_SOCKET (-3): wrong socket type.

Constructor & Destructor Documentation

Socket::Socket ( int  type = SOCK_STREAM)

Creates a new socket. Creates a AF_INET socket using the IPv4 Internet protocol. Type can be:

  • SOCK_STREAM (the default) for TCP/IP connected stream sockets
  • SOCK_DGRAM for UDP datagram sockets
