Detailed Description

TCP/IP or Datagram INET socket server.

Constructor & Destructor Documentation

ServerSocket::ServerSocket ( int  type = SOCK_STREAM)

creates a new ServerSocket. Creates a listening AF_INET socket (using the IPv4 Internet protocol) that waits for connection requests by client sockets. 'type' can be:

  • SOCK_STREAM (the default) for TCP/IP connected sockets
  • SOCK_DGRAM for UDP datagram sockets
